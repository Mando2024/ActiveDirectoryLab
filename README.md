<h1>Active Directory Home Lab</h1>


<h2>Description</h2>
In this lab I go over creating a new domain controller and example Windows devices, along with creating example GPO's for my new domain environment. The environment was created by using VMWare Workstation Player 17 
<br />


<h2>Environments Used </h2>

- <b> Two Windows 10 Pro VM's</b>
- <b>Windows Server 2022 R2</b>
